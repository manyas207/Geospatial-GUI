## Analysis

### `POST /api/query`

Upload GeoTIFFs (and optional shapefile parts) with a natural-language question. Runs LST or OBIA pipeline.

**Content-Type:** `multipart/form-data`

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `question` | string | yes | e.g. "What is the land surface temperature?" |
| `files` | file[] | yes | At least one `.tif`; OBIA may include `.shp`, `.shx`, `.dbf` |

**Response:** `QueryResponse`

```json
{
  "model": "lst",
  "summary": "Routed to LST based on: ...",
  "stats": { "min_C": 18.2, "max_C": 42.1, "mean_C": 31.5 },
  "logs": "...",
  "artifacts": [
    {
      "id": "...",
      "label": "LST output map",
      "filename": "LST_....tif",
      "kind": "geotiff",
      "download_url": "/api/artifacts/.../download",
      "preview_url": "/api/artifacts/.../preview"
    }
  ],
  "reference_layers": []
}
```

**Routing:** `backend/nlu.py` uses Ollama when `OLLAMA_ENABLED=true`; falls back to keywords if Ollama is unreachable.

## Follow-up chat

### `POST /api/followup`

Ask Ollama about the current dashboard or city context.

**Content-Type:** `application/json`

```json
{
  "question": "What was the mean temperature?",
  "context": {
    "model": "lst",
    "summary": "...",
    "stats": {},
    "logs": "",
    "raster": "scene.tif",
    "artifacts": [],
    "reference_layers": []
  }
}
```

**Response:**

```json
{ "answer": "The mean land surface temperature was 31.5°C." }
```

## City layers (Heat & Equity)

### `POST /api/city-layers`

Load census tract boundaries, ACS demographics, and map previews for a US city/county.

**Content-Type:** `application/json`

```json
{ "address": "Atlanta, GA" }
```

**Requires:** `CENSUS_API_KEY` in environment.

**Response:** `CityLayersResponse`

```json
{
  "address": "Atlanta, GA",
  "matched_address": "Atlanta, GA (city centroid)",
  "geocode": {
    "state_fips": "13",
    "county_fips": "121",
    "county_name": "Fulton County",
    "latitude": 33.749,
    "longitude": -84.388
  },
  "summary": {
    "tract_count": 327,
    "county": "Fulton County",
    "state": "Georgia",
    "total_population": 1066710,
    "median_income_usd": 64100,
    "avg_density_per_km2": 3842.1
  },
  "map_layers": {
    "density": {
      "id": "density",
      "label": "Population density",
      "field": "population_density_per_km2",
      "preview_url": "/api/city-layers/map/{token}/preview"
    },
    "income": { "...": "..." },
    "ethnicity": { "...": "..." },
    "tracts": { "...": "..." }
  },
  "worldpop": {
    "preview_url": "/api/city-layers/worldpop/{token}/preview",
    "bounds_wgs84": [-84.6, 33.5, -84.2, 33.9],
    "source": "WorldPop 2020 constrained (USA)"
  },
  "sources": {
    "geocoder": "Census Geocoder API",
    "tracts": "Census TIGER shapefile (cached)",
    "demographics": "Census ACS 5-year (13-121)",
    "map_render": "Server-side PNG choropleth (matplotlib + geopandas)",
    "gridded_population": "WorldPop 2020 constrained (USA)"
  }
}
```

### `GET /api/city-layers/map/{token}/preview`

Returns PNG choropleth for tracts (density, income, ethnicity, or boundaries).

### `GET /api/city-layers/worldpop/{token}/preview`

Returns PNG preview of WorldPop gridded population clipped to county bounds.

## Reference layers (Dashboard)

### `GET /api/reference-layers`

List GeoTIFFs found in `REFERENCE_DATA_DIR`.

**Response:** array of `ReferenceLayer` objects with `preview_url`, `download_url`, and `stats`.

### `GET /api/reference-layers/{layer_id}/preview`

PNG preview of a reference GeoTIFF.

### `GET /api/reference-layers/{layer_id}/download`

Download the original GeoTIFF.

---

## Pipeline artifacts

### `GET /api/artifacts/{artifact_id}/download`

Download LST/OBIA output file (GeoTIFF or GeoPackage).

### `GET /api/artifacts/{artifact_id}/preview`

PNG preview of a raster artifact.

Artifact IDs are opaque tokens encoding paths under `data/` (see `backend/artifacts.py`).

---

## Error responses

FastAPI returns JSON:

```json
{ "detail": "Human-readable error message" }
```

Common cases:

| Status | Cause |
|--------|-------|
| 400 | Missing files, bad address, invalid artifact token |
| 502 | External API unreachable (geocoder, census) |
| 500 | Pipeline or unexpected server error |
